环境变量设置

  • 所谓说的环境变量，React 指的是项目的接口方面设置。

  • 一般来说项目接口分为

    • 测试接口
    • 正式接口

  • 这时候就需要判断是开发模式还是线上模式

意义

  • 避免意外将缓慢的开发构建部署到生产环境中

操作流程

  • 新建一个文件 (类如：RecordsApi.js) 在一个专门的文件夹下面
  • 而文件中收纳了所有的 API ,统一管理。
  • 采取的 json server 方法
    • 案例中 实际json文件访问路径为 http://localhost:3004/recorsd

注意点

  • 必须以 REACT_APP_ 开头
  • 下面对应的 命令 执行的优先顺序
  • npm start：.env.development.local，.env.development，.env.local，.env
  • npm run build：.env.production.local，.env.production，.env.local，.env
  • npm test：.env.test.local，.env.test，.env（注.env.local丢失）
